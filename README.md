# backbone-history

  Backbone.History [component](https://github.com/component/component/wiki/Components)

## Installation

    $ component install spini/backbone-history

## Example

```js

var history = require('spini-backbone-history');

$(function(){
  ...
  history.start({pushState: true});
});

```
